Q：tensorflow 不能用 tensorboard 报出 “address is in use” 怎么解决？
A：using ps aux | grep tensorboard and kill are the ways I would do it

Q: Linux环境下怎么运行cuda程序？
A:
获取硬件平台设备属性 CPU properties：cat /proc/cpuinfo
Memory properties: cat /proc/meminfo
GPUproperties: nvcc devicepro.cu ./a.out

Q: Linux下编译cuda程序
A: 可以编译有图形界面的程序
nvcc julia_gpu.cu -lglut -lGLU -lGL

Q：如何显示文件夹的大小？
du -h 目录名 #显示该目录的大小
如 [root@localhost ~]# du -h /tmp
1.6M /tmp
Q: 在不知道搜哪个手册的情况下如何使用man命令？
A:
$ man -k password #这个就是查找关于password的命令

Q: How to clone codes from github respository?
A: git clone https://github.com/your-username/your-repository

Q: How to list trees in Linux?
A: ls -R
