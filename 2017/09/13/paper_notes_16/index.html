Abstract

  • dual pathway, 11-layer deep, 3D CNN for lesion segmentation.
  • Use image patches to adapte class imbalance.
  • Analyze the development of deeper cnn
  • For post processing, used a 3D fully connected conditional random field which effectively removes false positives.

2. Method

分割任务包含两个：
第一，3D CNN 提供一个高精度的segmentation map；
第二，CNN输出层的 fully connected 3D CRF（附加正则项）提供最终的分割标签。
